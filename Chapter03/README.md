## 栈
### 实现
栈是一个表，因此任何实现表的方法都能实现栈。
### 应用
1. 符号平衡
2. 后缀（逆波兰）表达式计算
3. 中缀到后缀的转换
4. 函数调用  
（代码实现了一个简单的计算器，应保证输入合法）
